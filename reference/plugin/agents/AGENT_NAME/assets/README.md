# assets/

Static files the agent uses or includes in its output.

Examples:
- Output templates the agent populates
- Images or icons embedded in produced files
- Configuration files the agent references

Reference from the agent's `SKILL.md`:

```markdown
Use the template at `${CLAUDE_PLUGIN_ROOT}/agents/AGENT_NAME/assets/template.md`.
```

Delete this directory if the agent uses no static files.
